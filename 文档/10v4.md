# 摘要

多模态会话情感识别需要在对话上下文中联合理解文本语义、语音韵律和视觉表情。现有 Transformer、图神经网络和提示学习方法能够提升跨模态建模能力，但多数方法仍将融合过程压缩为隐式高维表示：模型给出情感类别，却难以说明它参考了哪些相似案例、为何信任某一模态，以及外部知识在多大程度上改变了多模态判断。针对这一问题，本文提出一种证据锚定认知校准框架（Evidence-Anchored Cognitive Calibration, EACC），构建“检索证据—认知解析—可靠性分配—质量校准”的可追溯决策链。具体而言，本文首先从训练集中检索与当前话语在文本上下文和音视频状态上相似的样本，以数据集内部证据约束大语言模型推理；随后将大语言模型输出解析为结构化认知变量，包括情感软分布、VAD 评价、模态提示、推理置信度和检索质量；进一步地，模型基于这些认知变量动态估计文本、语音、视觉和认知推理四类信息源的可靠性，并通过质量感知校准门控控制认知信息对多模态表示的修正幅度。与此同时，本文利用 VAD 空间构造情感语义约束，使表示学习兼顾离散类别判别和连续情感结构。实验在 IEMOCAP 和 MELD 数据集上进行，结果表明，本文方法能够在保持有竞争力识别性能的同时，提供更清晰的证据来源、模态贡献和推理注入强度，从而提升多模态会话情感识别的机制可解释性与可信度。

# 引言

多模态会话情感识别旨在根据对话中的文本内容、语音韵律、视觉表情和上下文交互关系，识别每个话语所表达的情感状态。该任务是情感计算、自然语言处理和可信人机交互中的重要问题，在智能客服、社交媒体理解、心理健康分析和具身智能交互等场景中具有广泛应用价值。与单句情感分类不同，会话情感识别通常面临三类困难：第一，情感表达具有强上下文依赖，同一句话在不同对话历史和说话人状态下可能呈现不同情绪；第二，文本、语音和视觉并非始终一致，讽刺、沉默、遮挡、噪声和个体表达差异都会造成模态冲突；第三，情感类别之间存在连续语义关系，例如 happy 与 excited、angry 与 frustrated、neutral 与 sad 往往只在唤醒度、控制感或情绪强度上存在细粒度差别。已有研究从说话人关系、情绪转移、图结构上下文和多模态交互等角度提升会话情感识别能力 [1][22][34][53]，但复杂场景下的“为什么这样预测”仍然难以回答。

近年来，基于 Transformer、图神经网络和多模态提示学习的方法显著推动了 MERC 发展。MPT 将多模态提示和混合对比学习引入 Transformer 框架 [70]，DQ-Former 通过动态模态优先级刻画样本级模态贡献差异 [76]，VEGA 利用视觉原型强化情感语义对齐 [47]，超图、关系图和跨模态语义图方法也进一步增强了会话结构建模能力 [39][43][45][51]。然而，这些方法大多仍依赖端到端学习得到的隐式融合权重。即使模型内部存在注意力或门控机制，其权重也往往缺少可验证的外部证据支撑。当模型面对低质量视觉、语音噪声或文本歧义时，研究者很难判断模型究竟是依据相似情感案例、某一模态线索，还是仅仅依赖数据偏置完成预测。这种机制不透明性限制了模型在高可信应用中的使用。

大语言模型展现出的上下文理解、情感解释和常识推理能力为解决上述问题提供了新机会。近期工作已经尝试利用 LLM 建模说话人特征、语音细节、心理状态和对话场景信息，以增强 ERC/MERC 中的高层语义理解能力 [16][31][41][46]。知识增强方法也表明，情绪常识、人格信息、情绪原因和情绪转移知识能够为复杂会话中的情感判断提供重要补充 [6][7][20][58]。然而，直接把 LLM 输出作为情感预测或后验解释并不充分。一方面，LLM 可能过度依赖通用知识，与特定数据集的标签边界和标注风格不一致；另一方面，后验生成的解释并不一定真实参与神经模型决策，仍无法解释模型内部融合过程；此外，当输入证据不足或检索样本质量较低时，LLM 推理可能引入新的噪声。因此，关键问题不是简单“接入 LLM”，而是如何让 LLM 推理受到任务数据约束，并以可度量、可筛选、可控的方式参与多模态决策。

本文围绕这一问题提出证据锚定认知校准框架 EACC。核心思想是将 LLM 从外部预测器转化为可解释的认知信号提供者：模型首先通过多模态 RAG 从训练集中检索相似样本，使 LLM 推理对齐数据集内部标签边界；然后将 LLM 输出解析为情感软分布、VAD 评价、模态提示、置信度和检索质量；最后将这些认知变量注入可靠性融合模块，由模型动态决定文本、语音、视觉和 LLM 认知分支的贡献，并通过校准强度控制认知推理对多模态表示的修正程度。这样，最终预测不再只是一个黑箱分类结果，而是可以沿着检索样本、认知变量、可靠性权重和校准门控逐层追溯。

本文的主要贡献如下。

1. 提出一种证据约束的多模态 RAG-LLM 认知推理机制。该机制联合文本上下文相似度和音视频统计相似度检索训练集样本，使 LLM 推理受到数据集内部案例约束，从而降低其偏离任务标签体系的风险。
2. 构建结构化认知变量，将 LLM 输出从自然语言解释转化为可计算信号。认知变量包含情感软分布、VAD appraisal、模态提示、置信度、检索质量和 VAD 指导向量，既能参与模型训练，也能支撑解释分析。
3. 设计证据锚定认知校准模块。模型根据认知变量动态分配文本、语音、视觉和 LLM 认知分支的可靠性，并通过质量感知校准强度控制认知表示对基础多模态表示的插值式修正，避免低质量推理无条件覆盖多模态判断。
4. 引入 VAD 引导的情感语义约束。本文将离散情感标签与 valence-arousal-dominance 连续空间结合，使模型在区分类别的同时保留情感之间的强度、极性和控制感结构，尤其适用于 happy/excited、angry/frustrated、neutral/sad 等易混淆类别。

# 相关工作

## 2.1 上下文与图结构会话情感识别

会话情感识别的核心在于建模上下文依赖、说话人交互和情绪状态变化。DualGATs 通过双图注意力网络分别建模 discourse-aware graph 和 speaker-aware graph [1]，Curriculum Learning Meets DAG 将有向无环图引入多模态情感识别并结合课程学习 [22]，Adaptive Graph Learning 从数据中自适应学习多模态会话图结构 [34]。近期研究进一步关注高阶关系和复杂交互，Graph Spectrum 从图频谱角度重新审视 MERC 中的上下文传播机制 [38]，Dynamic Interactive Bimodal Hypergraph Networks 使用超图建模双模态高阶关系 [39]，Hybrid Relational Graphs 和 MATCH 分别从情感语义对齐和模态校准超图角度增强多模态融合 [43][45]。这些方法说明结构化上下文对 ERC/MERC 至关重要，但多数方法仍主要在神经特征或图拓扑内部学习关系，缺少可追溯外部证据。

## 2.2 多模态融合与可靠性建模

文本、语音和视觉在不同话语中的可靠性并不相同。CMCF-SRNet 通过跨模态上下文融合和语义细化提升模态交互质量 [3]，MultiEMO 利用相关性感知注意力框架建模多模态关系 [4]，Joyful 将联合模态融合与图对比学习结合起来 [17]。近期研究进一步关注模态平衡、动态优先级和可信输出，MERC Calibration 关注多模态情感识别中的校准问题 [73]，Ada2I 缓解模态不平衡 [74]，DQ-Former 根据样本动态查询不同模态优先级 [76]，DRKF 通过表示解耦和知识融合增强多模态情感识别 [77]。与这些方法相比，本文不仅学习隐式模态权重，还将 LLM 生成的模态提示、置信度、VAD 评价和检索质量显式纳入可靠性分配，使融合权重具备可解释来源。

## 2.3 知识增强、提示学习与 LLM 情感推理

会话情感往往依赖隐含常识、情绪原因、说话人特征和情绪转移规律。ECoK 挖掘情绪常识知识图谱 [6]，EmoTransKG 利用情绪知识图谱揭示情绪转移 [7]，SKIER 将符号知识融入会话情感识别 [32]，MKE-IGN 通过多知识增强交互图网络融合外部知识和多模态上下文 [21]，UniMEEC 将情感识别与情感原因分析统一建模 [20]。随着 LLM 发展，提示学习和 LLM 适配方法也被引入 MERC。MSE-Adapter 设计轻量插件赋予 LLM 多模态情感分析能力 [41]，LaERC-S 利用 LLM 挖掘说话人特征 [31]，Beyond Silent Letters 强调语音细节对 LLM 情感理解的重要性 [16]，DialogueMMT 利用对话场景理解增强多模态多任务调优 [11]。这些方法证明 LLM 可以提供高层语义补充；本文进一步强调，LLM 推理需要被训练集证据约束，并以结构化认知变量的形式真实进入模型内部计算链条。具体而言，本文不是将 LLM 作为独立分类器或后验解释器，而是把 RAG 检索证据、VAD 评价、模态可靠性提示和软标签分布编码为可学习认知特征，使其参与神经模型的前向融合、蒸馏监督和可靠性约束。

## 2.4 情感语义约束与可信 MERC

除准确率之外，MERC 还需要面对类别语义相近、模态缺失、样本不均衡和模型可信度不足等问题。Emotion-Anchored Contrastive Learning 使用情感锚点约束表示空间 [13]，Dual Contrastive Learning Framework 通过双重对比学习增强多模态表示 [30]，PEMV 利用近邻情绪均值向量改善情感空间分布 [15]。缺失模态和说话人泛化也是近期重要方向 [8][48][57][61][64]。本文与上述方向互补：通过 RAG 证据、认知变量、可靠性门控和校准强度，把性能提升与机制解释连接起来，使模型不仅能预测情感类别，还能回答“依据什么证据、信任哪些模态、LLM 推理产生了多大影响”。

# 方法

## 3.1 任务定义与总体框架

给定一个包含 $N$ 个话语的多模态对话：

$$
D=\{u_1,u_2,\ldots,u_N\}.
$$

其中，$D$ 表示一段完整对话；$N$ 表示该对话中的话语数量；$u_i$ 表示第 $i$ 个话语，$i\in\{1,\ldots,N\}$。

每个话语包含文本、语音和视觉三类模态：

$$
u_i=\{x_i^t,x_i^a,x_i^v\}.
$$

其中，$x_i^t$ 表示第 $i$ 个话语的文本特征，$x_i^a$ 表示语音特征，$x_i^v$ 表示视觉特征；上标 $t,a,v$ 分别对应 text、audio 和 visual。

模型目标是在对话上下文中预测话语级情感标签：

$$
\hat{y}_i=f(x_i^t,x_i^a,x_i^v,D),\quad \hat{y}_i\in\{1,\ldots,C\}.
$$

其中，$f(\cdot)$ 表示待学习的多模态情感识别模型；$\hat{y}_i$ 表示模型对话语 $u_i$ 的预测情感类别；$C$ 表示数据集中的情感类别数，IEMOCAP 中 $C=6$，MELD 中 $C=7$。

传统多模态融合通常直接学习如下映射：

$$
(x_i^t,x_i^a,x_i^v)\rightarrow h_i^{base}\rightarrow \hat{y}_i.
$$

其中，$h_i^{base}$ 表示由基础多模态骨干网络得到的隐式融合表示。该表示可以用于分类，但其内部证据来源、模态贡献和推理依据难以追踪。

本文将上述过程扩展为一条证据约束的认知决策链：

$$
\text{Multimodal Input}\rightarrow h_i^{base},\quad
\text{Retrieved Evidence}\rightarrow c_i,\quad
(h_i^{base},c_i)\rightarrow h_i^{final}\rightarrow \hat{p}_i.
$$

其中，$c_i$ 表示由 RAG-LLM 分支生成的结构化认知变量；$h_i^{final}$ 表示经过 EACC 校准后的最终表示；$\hat{p}_i\in\mathbb{R}^{C}$ 表示最终情感概率分布，$\hat{y}_i=\arg\max_k\hat{p}_{ik}$。

本文方法包含四个阶段。第一，三模态交互编码器从文本、语音和视觉中学习基础多模态表示。第二，多模态 RAG 从训练集中检索与当前话语相似的标注样本，为 LLM 提供数据集内部证据。第三，LLM 将证据和当前上下文解析为结构化认知变量，包括情感软分布、VAD appraisal、模态提示、置信度和检索质量。第四，模型根据认知变量计算可靠性权重和校准强度，以质量受控的方式修正基础多模态表示。

该框架的创新点在于，LLM 不作为外部分类器直接替代神经模型，也不只生成预测后的自然语言解释；相反，LLM 被转化为可计算、可监督、可筛选的认知信号，并通过可靠性门控和质量校准真实参与模型内部决策。

## 3.2 三模态交互编码器

本文使用预处理好的三类模态特征。文本特征维度为 1024，视觉 denseface 特征维度为 342；语音特征在 IEMOCAP 上为 1582 维，在 MELD 上为 300 维。为消除模态维度差异，模型首先通过一维卷积把三类模态投影到统一隐空间：

$$
h_i^m=\phi_m(x_i^m),\quad m\in\{t,a,v\}.
$$

其中，$m$ 表示模态索引，取值为 $t$、$a$ 或 $v$；$\phi_m(\cdot)$ 表示第 $m$ 个模态的一维卷积投影函数；$h_i^m\in\mathbb{R}^{d}$ 表示投影后的模态表示；$d$ 表示统一隐空间维度，本文实现中 $d=1024$。

随后加入位置编码和说话人嵌入：

$$
\tilde{h}_i^m=h_i^m+p_i+s_i.
$$

其中，$\tilde{h}_i^m$ 表示加入对话位置信息和说话人信息后的模态表示；$p_i\in\mathbb{R}^{d}$ 表示第 $i$ 个话语的位置编码；$s_i\in\mathbb{R}^{d}$ 表示该话语对应说话人的嵌入向量。IEMOCAP 设置 2 个说话人嵌入，MELD 设置 9 个说话人嵌入。

为了同时建模模态内上下文和跨模态交互，本文构建九条 Transformer 分支：

$$
z_i^{m\leftarrow n}=\operatorname{Transformer}_{m\leftarrow n}(\tilde{H}^n,\tilde{H}^m),
\quad m,n\in\{t,a,v\}.
$$

其中，$\tilde{H}^m=[\tilde{h}_1^m,\ldots,\tilde{h}_N^m]$ 表示整段对话在模态 $m$ 上的序列表示；$z_i^{m\leftarrow n}$ 表示目标模态 $m$ 从源模态 $n$ 接收信息后，在第 $i$ 个话语处得到的交互表示；当 $m=n$ 时，该分支执行模态内上下文建模；当 $m\neq n$ 时，该分支执行跨模态注意力交互。九条分支分别对应三种目标模态和三种源模态的两两组合。

对于每个目标模态，模型拼接来自三个源模态的交互结果，并降维得到目标模态增强表示：

$$
r_i^m=W_m[z_i^{m\leftarrow t};z_i^{m\leftarrow a};z_i^{m\leftarrow v}]+b_m.
$$

其中，$r_i^m\in\mathbb{R}^{d}$ 表示目标模态 $m$ 的增强表示；$[\cdot;\cdot;\cdot]$ 表示向量拼接；$W_m\in\mathbb{R}^{d\times 3d}$ 和 $b_m\in\mathbb{R}^{d}$ 是可学习线性变换参数。

基础多模态表示由门控融合得到：

$$
h_i^{base}=\operatorname{Gate}(r_i^t,r_i^a,r_i^v).
$$

其中，$\operatorname{Gate}(\cdot)$ 表示根据三类增强表示自适应计算融合权重的门控函数；$h_i^{base}\in\mathbb{R}^{d}$ 表示不引入 LLM 认知变量时的基础多模态表示。该表示是后续 EACC 校准的起点，而不是被 LLM 分支替代。

## 3.3 多模态 RAG 证据构建

LLM 的通用情感知识并不天然等同于 IEMOCAP 或 MELD 的标签边界。为使 LLM 推理对齐目标数据集，本文从训练集中检索相似标注样本作为证据。对目标话语 $u_i$ 和候选训练话语 $u_j$，检索得分定义为：

$$
s(u_i,u_j)=\lambda s_{text}(u_i,u_j)+(1-\lambda)s_{av}(u_i,u_j).
$$

其中，$s(u_i,u_j)$ 表示目标话语 $u_i$ 与候选话语 $u_j$ 的综合相似度；$s_{text}$ 表示文本上下文相似度；$s_{av}$ 表示音视频状态相似度；$\lambda\in[0,1]$ 是文本检索权重，本文实验中 $\lambda=0.7$。

文本相似度采用 TF-IDF 余弦相似度：

$$
s_{text}(u_i,u_j)=
\frac{\mathbf{q}_i^\top\mathbf{q}_j}
{\|\mathbf{q}_i\|_2\|\mathbf{q}_j\|_2}.
$$

其中，$\mathbf{q}_i$ 和 $\mathbf{q}_j$ 分别表示话语 $u_i$ 和 $u_j$ 的上下文 TF-IDF 向量；$\top$ 表示向量内积；$\|\cdot\|_2$ 表示二范数。实现中使用 1-2 gram TF-IDF，最大特征数为 50000。

音视频统计向量定义为：

$$
e_i^{av}=[E_i^a,\sigma_i^a,P_i^a,\Delta_i^a,E_i^v,\sigma_i^v,P_i^v,\Delta_i^v].
$$

其中，$e_i^{av}\in\mathbb{R}^{8}$ 表示第 $i$ 个话语的音视频统计向量；$E$ 表示平均能量；$\sigma$ 表示标准差或变化程度；$P$ 表示峰值强度；$\Delta$ 表示与前一话语的变化幅度；上标 $a$ 和 $v$ 分别表示 audio 和 visual。

统计向量标准化后计算音视频状态相似度：

$$
s_{av}(u_i,u_j)=
\frac{\bar{e}_i^{av\top}\bar{e}_j^{av}}
{\|\bar{e}_i^{av}\|_2\|\bar{e}_j^{av}\|_2}.
$$

其中，$\bar{e}_i^{av}$ 和 $\bar{e}_j^{av}$ 表示标准化后的音视频统计向量；$s_{av}$ 越大，说明两个话语在语音强度、视觉变化和相邻话语偏移等状态上越相似。

最终选取得分最高的 $K$ 个训练样本作为检索证据：

$$
\mathcal{R}_i=\operatorname{TopK}_{u_j\in\mathcal{D}_{train}}s(u_i,u_j).
$$

其中，$\mathcal{R}_i$ 表示目标话语 $u_i$ 的检索证据集合；$\mathcal{D}_{train}$ 表示训练集；$K$ 表示检索样本数量，本文实验中 $K=5$；$\operatorname{TopK}$ 表示选取得分最高的 $K$ 个候选样本。

为了衡量检索证据是否可靠，本文定义检索质量：

$$
\rho_i=0.35s_i^{top}+0.20s_i^{mean}+0.20\max(s_i^{mm},0)+0.25a_i^{label}.
$$

其中，$\rho_i\in[0,1]$ 表示第 $i$ 个话语的 RAG 检索质量；$s_i^{top}$ 表示检索集合中的最高综合相似度；$s_i^{mean}$ 表示检索集合的平均综合相似度；$s_i^{mm}$ 表示检索集合的平均音视频相似度；$a_i^{label}$ 表示检索样本标签与 LLM 预测标签的一致比例。$\rho_i$ 越高，说明检索证据越集中、越可信，后续认知校准可被赋予更高权重。

## 3.4 结构化认知变量

LLM 接收当前话语、上下文、音视频统计和检索证据后，被要求返回一个 JSON 对象。该对象包含 `label`、`confidence`、`rationale`、`appraisal` 和 `modality_hint`。与自由文本解释相比，JSON 输出可以被稳定解析为向量，并参与模型训练。

认知特征定义为：

$$
c_i=[p_i^{llm};a_i;q_i;\gamma_i;\rho_i;g_i].
$$

其中，$c_i$ 表示第 $i$ 个话语的完整认知特征；分号表示向量拼接；$p_i^{llm}$ 是 LLM 情感软分布；$a_i$ 是 VAD appraisal；$q_i$ 是模态提示；$\gamma_i$ 是 LLM 置信度；$\rho_i$ 是检索质量；$g_i$ 是 VAD guidance。

LLM 情感软分布为：

$$
p_i^{llm}=[p_{i1}^{llm},p_{i2}^{llm},\ldots,p_{iC}^{llm}],\quad
\sum_{k=1}^{C}p_{ik}^{llm}=1.
$$

其中，$p_{ik}^{llm}$ 表示 LLM 认为话语 $u_i$ 属于第 $k$ 个情感类别的概率；$C$ 是类别数。若 LLM 只输出 hard label，本文根据置信度 $\gamma_i$ 构造平滑分布，使预测类别概率为 $\gamma_i$，其他类别均分剩余概率。

VAD appraisal 定义为：

$$
a_i=[v_i,\alpha_i,d_i,z_i].
$$

其中，$v_i$ 表示 valence，即情感正负性；$\alpha_i$ 表示 arousal，即唤醒度；$d_i$ 表示 dominance，即控制感；$z_i$ 表示 certainty，即 appraisal 的确定性。

模态提示定义为：

$$
q_i=[q_i^t,q_i^a,q_i^v],\quad q_i^t+q_i^a+q_i^v=1.
$$

其中，$q_i^t$、$q_i^a$ 和 $q_i^v$ 分别表示 LLM 认为文本、语音和视觉在当前话语中的重要性。该提示不是最终融合权重，而是后续可靠性门控的外部认知先验。

VAD guidance 由 appraisal 派生：

$$
g_i=[g_i^{pos},g_i^{ctrl},g_i^{low}],
$$

$$
g_i^{pos}=\max(v_i,0)\cdot \alpha_i,
$$

$$
g_i^{ctrl}=\max(-v_i,0)\cdot\max(\alpha_i,0)\cdot d_i,
$$

$$
g_i^{low}=\max(-v_i,0)\cdot\max(-\alpha_i,0).
$$

其中，$g_i^{pos}$ 表示正向高激活因素，适合描述 happy、excited 等情绪；$g_i^{ctrl}$ 表示负向高唤醒且带控制感的因素，适合区分 angry、frustrated 等情绪；$g_i^{low}$ 表示负向低唤醒因素，适合描述 sad 等情绪。$\max(\cdot,0)$ 用于只保留对应方向上的有效情感强度。

因此，认知特征维度为：

$$
\dim(c_i)=C+4+3+1+1+3=C+12.
$$

其中，$C$ 来自 LLM 情感软分布；4 来自 VAD appraisal；3 来自模态提示；两个 1 分别来自置信度和检索质量；最后 3 来自 VAD guidance。IEMOCAP 中 $\dim(c_i)=18$，MELD 中 $\dim(c_i)=19$。

## 3.5 证据锚定认知校准

证据锚定认知校准（EACC）是本文的核心融合模块。该模块不把 LLM 输出直接作为最终预测，而是把结构化认知变量转化为两个可解释控制量：一是文本、语音、视觉和认知分支之间的可靠性分布，二是认知分支对基础多模态表示的校准强度。前者回答模型“信任哪些信息源”，后者回答模型“接受多少外部认知修正”。

首先将认知特征投影到与多模态表示一致的隐空间：

$$
h_i^c=\psi(c_i).
$$

其中，$h_i^c\in\mathbb{R}^{d}$ 表示认知分支隐表示；$\psi(\cdot)$ 表示认知投影网络，由线性层、ReLU 和 LayerNorm 组成；$d=1024$ 是统一隐空间维度。

模型随后计算四类信息源的基础可靠性 logits：

$$
o_i=W_r[r_i^t;r_i^a;r_i^v;h_i^c]+b_r.
$$

其中，$o_i\in\mathbb{R}^{4}$ 表示四个信息源的未归一化可靠性分数；$W_r\in\mathbb{R}^{4\times 4d}$ 和 $b_r\in\mathbb{R}^{4}$ 是可学习参数；四个信息源依次为文本、语音、视觉和 LLM 认知分支。

为使可靠性门控受到认知变量显式影响，本文构造模态提示偏置：

$$
b_i^{hint}=W_h[q_i^t,q_i^a,q_i^v,\tau_i],
$$

$$
\tau_i=\gamma_i(0.5+0.5\rho_i).
$$

其中，$b_i^{hint}\in\mathbb{R}^{4}$ 表示由模态提示产生的门控偏置；$W_h\in\mathbb{R}^{4\times4}$ 是可学习参数；$\tau_i$ 表示 reasoning trust，即 LLM 认知分支可信度；$\gamma_i$ 越高且 $\rho_i$ 越高，$\tau_i$ 越大。

同时构造 VAD 偏置：

$$
b_i^{vad}=W_v[a_i;g_i].
$$

其中，$b_i^{vad}\in\mathbb{R}^{4}$ 表示由 VAD appraisal 和 VAD guidance 产生的门控偏置；$W_v\in\mathbb{R}^{4\times7}$ 是可学习参数；$[a_i;g_i]\in\mathbb{R}^{7}$ 由 4 维 appraisal 和 3 维 guidance 拼接得到。

最终可靠性权重为：

$$
\beta_i=\operatorname{softmax}(o_i+b_i^{hint}+b_i^{vad}).
$$

其中，$\beta_i=[\beta_i^t,\beta_i^a,\beta_i^v,\beta_i^c]\in\mathbb{R}^{4}$；$\beta_i^t$、$\beta_i^a$、$\beta_i^v$ 和 $\beta_i^c$ 分别表示文本、语音、视觉和认知分支的可靠性权重；$\operatorname{softmax}$ 保证四个权重非负且和为 1。

若某个话语不存在 LLM cache，或其检索质量和置信度未通过阈值，认知分支会被 mask。记 mask 为：

$$
M_i^{llm}=\mathbb{I}(\rho_i\ge\delta_q)\cdot\mathbb{I}(\gamma_i\ge\delta_c).
$$

其中，$M_i^{llm}\in\{0,1\}$ 表示 LLM 认知分支是否有效；$\mathbb{I}(\cdot)$ 是指示函数；$\delta_q$ 是检索质量阈值；$\delta_c$ 是置信度阈值。IEMOCAP 使用 $\delta_q=0.60,\delta_c=0.85$，MELD 使用 $\delta_q=0.65,\delta_c=0.90$。

可靠性感知融合表示为：

$$
h_i^{rel}=\beta_i^t r_i^t+\beta_i^a r_i^a+\beta_i^v r_i^v+\beta_i^c h_i^c.
$$

其中，$h_i^{rel}\in\mathbb{R}^{d}$ 表示可靠性感知融合表示；$\beta_i^c h_i^c$ 是认知分支的贡献项；当 $M_i^{llm}=0$ 时，模型抑制 $\beta_i^c$，避免低质量认知信息进入融合。

为了避免 LLM 推理直接覆盖多模态判断，本文采用质量感知的插值校准：

$$
h_i^{final}=h_i^{base}+\eta_i(h_i^{rel}-h_i^{base}).
$$

其中，$h_i^{final}$ 表示最终用于分类的表示；$h_i^{rel}-h_i^{base}$ 表示可靠性融合结果相对于基础多模态表示的校准方向；$\eta_i\in[0,1]$ 表示校准强度，决定模型接受多少认知修正。

校准强度定义为：

$$
\eta_i=\sigma(\theta_\eta)\cdot(0.5+\sigma(W_\eta[\gamma_i;\rho_i]+b_\eta))\cdot \gamma_i(0.5+0.5\rho_i)\cdot M_i^{llm}.
$$

其中，$\sigma(\cdot)$ 表示 sigmoid 函数；$\theta_\eta$ 是可学习全局校准参数；$W_\eta\in\mathbb{R}^{1\times2}$ 和 $b_\eta\in\mathbb{R}$ 是校准控制器参数；$[\gamma_i;\rho_i]$ 是置信度和检索质量拼接向量；$M_i^{llm}$ 保证未通过质量门控的认知信息不会产生表示校准。实现中 `llm-residual-init` 会被限制在 $[10^{-4},0.99]$，从而避免初始校准幅度过大或完全为零。

最终分类概率为：

$$
\hat{p}_i=\operatorname{softmax}(W_o h_i^{final}+b_o).
$$

其中，$W_o\in\mathbb{R}^{C\times d}$ 和 $b_o\in\mathbb{R}^{C}$ 是分类器参数；$\hat{p}_i=[\hat{p}_{i1},\ldots,\hat{p}_{iC}]$ 是最终情感概率分布；$\hat{p}_{ik}$ 表示话语 $u_i$ 属于第 $k$ 类情感的概率。

该模块的创新性在于，LLM 认知信息同时影响“信任谁”和“改多少”：可靠性权重 $\beta_i$ 决定文本、语音、视觉和认知分支的相对贡献，校准强度 $\eta_i$ 决定认知融合结果对基础多模态表示的修正幅度。因此，模型既能吸收 LLM 的认知推理，又能在推理质量不足时回退到多模态编码器判断。

## 3.6 训练目标

设一个 mini-batch 中所有有效话语索引集合为：

$$
\Omega=\{i\mid m_i=1\}.
$$

其中，$\Omega$ 表示非 padding 话语集合；$m_i\in\{0,1\}$ 是有效话语 mask，$m_i=1$ 表示真实话语，$m_i=0$ 表示 padding；$|\Omega|$ 表示有效话语数量。

本文将所有优化项组织为三类联合目标：监督判别目标、认知对齐目标和情感几何目标。总损失写为：

$$
\mathcal{L}
=\mathcal{L}_{sup}
+\lambda_{cog}\mathcal{L}_{cog}
+\lambda_{aff}\mathcal{L}_{aff}.
$$

其中，$\mathcal{L}_{sup}$ 负责标签监督下的情感判别，$\mathcal{L}_{cog}$ 负责将高质量 LLM 认知信号与模型预测和可靠性门控对齐，$\mathcal{L}_{aff}$ 负责保持表示空间中的连续情感语义结构。$\lambda_{cog}$ 和 $\lambda_{aff}$ 分别控制认知对齐和情感几何约束的贡献，在实现中对应 LLM 蒸馏、可靠性监督和 VAD 对比项的实验权重。

### 3.6.1 监督判别目标

监督判别目标同时约束最终融合分支、三个单模态分支，并使用融合分支的软分布蒸馏单模态分支：

$$
\mathcal{L}_{sup}
=\mathcal{L}_{cls}
+\lambda_{uni}\mathcal{L}_{uni}
+\lambda_{sd}\mathcal{L}_{sd}.
$$

最终分类损失为：

$$
\mathcal{L}_{cls}
=-\frac{1}{Z_{ce}}\sum_{i\in\Omega}\omega_{y_i}\log \hat{p}_{i,y_i}.
$$

其中，$y_i$ 是话语 $u_i$ 的真实情感标签；$\hat{p}_{i,y_i}$ 是模型预测为真实类别的概率；$\omega_{y_i}$ 是类别权重；$Z_{ce}=\sum_{i\in\Omega}\omega_{y_i}$ 是归一化项。IEMOCAP 使用类别权重缓解类别不均衡，MELD 默认不额外使用类别权重，此时 $\omega_{y_i}=1$。单模态监督项定义为：

$$
\mathcal{L}_{uni}
=-\frac{1}{Z_{ce}}\sum_{i\in\Omega}\omega_{y_i}
\left(
\log p_{i,y_i}^{t}
+\log p_{i,y_i}^{a}
+\log p_{i,y_i}^{v}
\right).
$$

其中，$p_i^t$、$p_i^a$ 和 $p_i^v$ 分别表示文本、语音和视觉单模态分支的预测分布；$p_{i,y_i}^{t}$ 表示文本分支对真实类别 $y_i$ 的预测概率，其余同理。该损失避免模型只依赖融合分支而忽略单模态判别能力。自蒸馏项使用最终融合分支的 softened distribution 约束三类单模态分支：

$$
\mathcal{L}_{sd}
=\frac{1}{|\Omega|}\sum_{i\in\Omega}
\left[
\operatorname{KL}(p_i^{all,\tau}\Vert p_i^{t,\tau})
+\operatorname{KL}(p_i^{all,\tau}\Vert p_i^{a,\tau})
+\operatorname{KL}(p_i^{all,\tau}\Vert p_i^{v,\tau})
\right].
$$

其中，$p_i^{all,\tau}$ 表示最终融合分支在温度 $\tau$ 下的 softened probability；$p_i^{t,\tau}$、$p_i^{a,\tau}$、$p_i^{v,\tau}$ 分别表示三类单模态分支在同一温度下的 softened probability；$\tau$ 为蒸馏温度；$\operatorname{KL}(P\Vert Q)=\sum_k P_k\log(P_k/Q_k)$ 表示 KL 散度。该损失使单模态分支学习融合分支中的软类别关系。

### 3.6.2 认知对齐目标

认知对齐目标把 LLM 的情感软分布和可靠性提示共同注入训练过程：

$$
\mathcal{L}_{cog}
=\lambda_{llm}\mathcal{L}_{llm}
+\lambda_{rel}\mathcal{L}_{rel}.
$$

LLM 认知蒸馏将结构化认知变量中的情感软分布注入最终分类器：

$$
\mathcal{L}_{llm}
=\frac{\sum_{i\in\Omega} w_i^{llm}
\operatorname{KL}(p_i^{llm}\Vert \hat{p}_i)}
{\max(\sum_{i\in\Omega} w_i^{llm},1)}.
$$

其中，$p_i^{llm}$ 是 LLM 生成或由 LLM label 转换得到的情感软分布；$\hat{p}_i$ 是模型最终预测分布；$w_i^{llm}$ 是 LLM 蒸馏样本权重。分母中的 $\max(\cdot,1)$ 用于数值保护，避免没有有效 LLM 样本时除零。

蒸馏权重定义为：

$$
w_i^{llm}=M_i^{cache}\cdot M_i^{llm}\cdot \rho_i\cdot \gamma_i.
$$

其中，$M_i^{cache}$ 表示该话语是否存在 LLM cache；$M_i^{llm}$ 表示该话语是否通过质量阈值和置信度阈值；$\rho_i$ 是检索质量；$\gamma_i$ 是 LLM 置信度。该权重使模型只从高质量、高置信度的 LLM 认知分布中学习。

为使可靠性门控与认知变量对齐，本文构造目标可靠性分布 $\beta_i^\ast$。首先由 VAD 构造模态先验：

$$
u_i^t=|z_i|+|g_i^{low}|+0.25\max(1-|r_i|,0),
$$

$$
u_i^a=|r_i|+|g_i^{ctrl}|,
$$

$$
u_i^v=0.25\max(v_i,0)+|g_i^{pos}|+0.10|d_i|.
$$

其中，$u_i^t$、$u_i^a$ 和 $u_i^v$ 分别表示由 VAD 推导出的文本、语音和视觉模态先验；$z_i$ 是 certainty；$\alpha_i$ 是 arousal；$v_i$ 是 valence；$d_i$ 是 dominance；$g_i^{pos}$、$g_i^{ctrl}$ 和 $g_i^{low}$ 是 VAD guidance 三个分量。

归一化得到 VAD 模态分布：

$$
\tilde{u}_i=\operatorname{Normalize}([u_i^t,u_i^a,u_i^v]).
$$

其中，$\tilde{u}_i\in\mathbb{R}^{3}$ 表示归一化后的 VAD 模态先验；$\operatorname{Normalize}$ 表示除以三个分量之和。

融合 LLM 模态提示和 VAD 模态先验：

$$
\mu_i=\operatorname{Normalize}(0.65q_i+0.35\tilde{u}_i).
$$

其中，$\mu_i=[\mu_i^t,\mu_i^a,\mu_i^v]$ 是文本、语音和视觉三类模态的目标分布；$q_i$ 是 LLM 输出的模态提示；0.65 和 0.35 是本文采用的固定融合系数。

认知分支目标权重为：

$$
\pi_i=\operatorname{clip}\left(\gamma_i(0.5+0.5\rho_i)(0.12+0.18s_i^{vad}),0,0.30\right),
$$

$$
s_i^{vad}=\operatorname{clip}\left(\frac{|v_i|+|r_i|+|d_i|}{3}
+\frac{|g_i^{pos}|+|g_i^{ctrl}|+|g_i^{low}|}{3},0,1\right).
$$

其中，$\pi_i$ 表示认知分支在目标可靠性分布中的先验权重；$s_i^{vad}$ 表示 VAD 强度；$\operatorname{clip}(x,l,u)$ 表示把 $x$ 限制在 $[l,u]$ 范围内。本文将 $\pi_i$ 上限限制为 0.30，避免目标分布过度偏向 LLM 认知分支。

最终目标可靠性分布为：

$$
\beta_i^\ast=
\operatorname{Normalize}([\mu_i^t(1-\pi_i),\mu_i^a(1-\pi_i),\mu_i^v(1-\pi_i),\pi_i]).
$$

其中，$\beta_i^\ast\in\mathbb{R}^{4}$ 表示文本、语音、视觉和认知分支的目标可靠性分布。

可靠性监督损失为：

$$
\mathcal{L}_{rel}
=\frac{\sum_{i\in\Omega}w_i^{rel}\|\beta_i-\beta_i^\ast\|_2^2}
{\max(\sum_{i\in\Omega}w_i^{rel},1)}.
$$

其中，$\beta_i$ 是模型预测的可靠性权重；$\beta_i^\ast$ 是目标可靠性分布；$\|\cdot\|_2^2$ 表示平方二范数；$w_i^{rel}=M_i^{llm}\operatorname{clip}(\rho_i\gamma_i,0,1)$ 是可靠性监督权重。

### 3.6.3 情感几何目标

情感几何目标利用 VAD 空间补充离散标签监督，使模型表示不仅区分情感类别，也保留类别之间的连续语义关系：

$$
\mathcal{L}_{aff}=\mathcal{L}_{vad}.
$$

设所有满足有效 mask 的话语表示集合为：

$$
\mathcal{V}=\{i\mid i\in\Omega,\ M_i^{cache}=1,\ M_i^{llm}=1\}.
$$

其中，$\mathcal{V}$ 表示参与 VAD 对比学习的有效样本集合；只有存在 LLM 认知特征且通过质量过滤的样本才参与该损失。

归一化表示为：

$$
\bar{h}_i=\frac{h_i^{final}}{\|h_i^{final}\|_2}.
$$

其中，$\bar{h}_i$ 是单位长度的最终表示，用于计算余弦相似度。

样本间相似度为：

$$
\operatorname{sim}_{ij}=\frac{\bar{h}_i^\top\bar{h}_j}{\tau_c}.
$$

其中，$\operatorname{sim}_{ij}$ 表示样本 $i$ 和样本 $j$ 的温度缩放相似度；$\tau_c$ 是对比学习温度，本文设为 0.2。

VAD 相似度为：

$$
\alpha_{ij}^{vad}=\exp(-\|a_i^{vad}-a_j^{vad}\|_2).
$$

其中，$a_i^{vad}=[v_i,r_i,d_i]$ 表示样本 $i$ 的前三维 VAD appraisal；$\alpha_{ij}^{vad}$ 越大，说明两个样本在连续情感空间中越接近。

同类正样本权重为：

$$
P_{ij}=\mathbb{I}(y_i=y_j)\mathbb{I}(i\neq j)(0.5+0.5\alpha_{ij}^{vad}).
$$

其中，$P_{ij}$ 表示样本 $j$ 作为样本 $i$ 正样本的权重；若二者标签不同或 $i=j$，则 $P_{ij}=0$；若标签相同且 VAD 更接近，则正样本权重更高。

正样本对比损失为：

$$
\mathcal{L}_{pos}
=-\frac{1}{|\mathcal{V}|}\sum_{i\in\mathcal{V}}
\frac{\sum_{j\in\mathcal{V}}P_{ij}
\log\frac{\exp(\operatorname{sim}_{ij})}
{\sum_{k\in\mathcal{V},k\neq i}\exp(\operatorname{sim}_{ik})}}
{\max(\sum_{j\in\mathcal{V}}P_{ij},1)}.
$$

其中，$\mathcal{L}_{pos}$ 拉近同类别且 VAD 相近的样本；分母中的求和项是对比学习中的候选样本归一化项；$\max(\sum_jP_{ij},1)$ 用于避免没有正样本时除零。

对于 IEMOCAP，本文额外定义三组易混淆负样本对：happy/excited、angry/frustrated 和 neutral/sad。负样本间隔损失为：

$$
\mathcal{L}_{neg}
=\frac{\sum_{(i,j)\in\mathcal{C}}
\max(\bar{h}_i^\top\bar{h}_j-m,0)
(0.5+0.5(1-\alpha_{ij}^{vad}))}
{\max(|\mathcal{C}|,1)}.
$$

其中，$\mathcal{C}$ 表示易混淆负样本对集合；$m$ 是间隔阈值，本文设为 0.2；当两个易混淆类别样本表示过近时，损失会增大；若它们在 VAD 空间也相距较远，则惩罚更强。

最终 VAD 对比损失为：

$$
\mathcal{L}_{vad}=\mathcal{L}_{pos}+\mathcal{L}_{neg}.
$$

其中，$\mathcal{L}_{pos}$ 强化同标签且 VAD 接近样本的聚合，$\mathcal{L}_{neg}$ 拉开易混淆情感对，二者共同使表示空间同时符合离散标签和连续情感语义。

## 3.7 可解释决策链

本文方法的解释性来自模型内部计算链条，而不是预测后的自然语言说明。对于任一话语 $u_i$，模型可以形成如下证据链：

$$
\mathcal{R}_i\rightarrow c_i\rightarrow \beta_i\rightarrow \eta_i\rightarrow \hat{p}_i.
$$

其中，$\mathcal{R}_i$ 表示检索证据集合；$c_i$ 表示结构化认知变量；$\beta_i$ 表示四类信息源的可靠性分布；$\eta_i$ 表示认知校准强度；$\hat{p}_i$ 表示最终情感概率分布。

这条链条可以被分解为三个解释层次。第一是证据层，研究者可以查看 $\mathcal{R}_i$ 中的 retrieved examples，判断模型参考的训练案例是否合理。第二是认知层，研究者可以检查 $c_i$ 中的 LLM label、confidence、VAD appraisal 和 modality hint，判断 LLM 对当前话语的认知解释是否合理。第三是融合层，研究者可以分析 $\beta_i$ 和 $\eta_i$，判断模型是否真正采纳了 LLM 认知信息，以及采纳程度是否与证据质量一致。相比单纯输出自然语言解释，该链条与模型实际计算过程一致，因此更适合作为机制可解释性分析依据。

## 4.2 实现细节

基础编码器采用三模态 Transformer 交互结构，隐藏维度为 1024，dropout 为 0.5，注意力头数为 8，权重衰减为 1e-5，训练轮数为 150。认知 cache 离线生成，默认 LLM 为 Qwen2.5-7B-Instruct；RAG 使用 $K=5$，上下文窗口为 3，文本检索权重为 0.7，生成温度为 0。训练阶段不加载 LLM，只读取 jsonl cache。

实验中，IEMOCAP 使用 `train.py`，batch size 为 16，学习率为 1e-4，自蒸馏温度为 1；MELD 使用 `train_1.py`，batch size 为 8，学习率为 5e-6，自蒸馏温度为 8。完整模型超参数如表 2 所示。

# 结论

本文针对多模态会话情感识别中融合过程难解释、模态贡献难追踪和 LLM 推理难控制的问题，提出证据锚定认知校准框架 EACC。该框架通过多模态 RAG 检索训练集内部相似案例，为 LLM 情感推理提供证据约束；同时将 LLM 输出解析为情感软分布、VAD 评价、模态提示、置信度和检索质量等结构化认知变量，并通过可靠性门控和质量感知校准机制注入多模态表示。实验结果表明，本文方法在 IEMOCAP 和 MELD 上保持有竞争力的识别性能，同时能够输出可追溯的证据链、模态可靠性和认知校准强度。未来工作可进一步扩展到更多 LLM backbone、缺失模态场景以及基于可靠性权重的交互式情感解释。
